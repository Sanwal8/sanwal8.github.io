Featured Projects

IHGAMP: HRD Prediction from Histopathology

Integrative histopathology-genomics framework predicting homologous recombination deficiency directly from H&E whole-slide images. Validated across 9 CPTAC cancer types with 5,607 slides and trained on 20,007 TCGA slides.

Tech: Python, PyTorch, OpenCLIP, Ridge Regression, Bootstrap CI

OpenSlideFM: Foundation Model for WSI Analysis

Self-supervised foundation model benchmarked against state-of-the-art models (UNI, Virchow2, CONCH) on CAMELYON16/17 and PANDA Gleason grading tasks using multiple instance learning architecture.

Tech: PyTorch, OpenSlide, MIL, Self-supervised Learning

COSMOS: ADC Target Prediction

AI framework for predicting TROP2 and NECTIN-4 expression from H&E slides to guide antibody-drug conjugate therapy selection. Incorporates proteomics supervision for enhanced accuracy.

Tech: Python, PyTorch, Multi-modal Learning, Proteomics

ICU Mortality Prediction

Machine learning pipeline predicting mortality risk in ICU patients using time-series analysis of vital signs and lab values, with focus on sepsis scenarios.

Tech: Python, XGBoost, scikit-learn, Time-series Analysis

Guard-Constrained Human-Robot Co-Navigation

Digital twin framework for safety-constrained navigation in warehouse fulfillment systems with real-time collision avoidance in dynamic environments. Submitted to IJPR Special Issue.

Tech: Python, ROS, Gazebo, TensorFlow, Digital Twin
